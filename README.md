# VisionIX

VisionIX is an Enterprise AI Analytics Platform that transforms raw datasets into validated data, analytics, machine learning models, explainable AI, dashboards, insights, and executive reports.

## Features
- Data validation and cleaning
- Feature engineering
- Exploratory data analysis
- KPI detection
- Agentic AutoML
- Model comparison
- Explainable AI
- Feature importance
- Interactive dashboard
- Executive report generation

## Tech Stack
Python, Streamlit, Pandas, NumPy, Scikit-learn, Plotly, SHAP, ReportLab

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
